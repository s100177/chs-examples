## 系统拓扑结构
水利系统的连接关系如下:

1. upstream_reservoir → gate_1
2. gate_1 → canal_1
3. canal_1 → gate_2
4. gate_2 → canal_2

这形成了一个完整的水流传输网络，水流从上游组件流向下游组件。
