# 配置文件指南

## 配置文件结构

### config.yml
主配置文件，包含仿真基础设置。

### agents.yml
智能体配置文件，定义系统中的智能体。

### components.yml
组件配置文件，定义系统组件。

### topology.yml
拓扑配置文件，定义组件间的连接关系。

## 扰动配置文件

- actuator_disturbance.yml: 执行器扰动配置
- sensor_disturbance.yml: 传感器扰动配置
- inflow_disturbance.yml: 入流扰动配置
- network_disturbance.yml: 网络扰动配置
